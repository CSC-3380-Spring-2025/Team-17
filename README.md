# How to Run Dev and Test Environment

## Dependencies
- NodeJS
- Expocli
### Downloading Dependencies

Verify you have Node.js installed

```sh
node -v #v22.15.0
```

```sh
npm -v #v10.9.2
```

Install Expo-cli

```sh
npm install expo-cli
```

Install Expo Go App
!!! ENSURE YOU ARE USING SDK 52 !!!

## Execution

```sh
npx expo start
```
